系统核心说明

本系统采用前后端分离架构，前端基于微信小程序实现用户交互，后端基于 Spring Boot 构建 RESTful 服务，数据层采用 MySQL 实现持久化存储。系统在满足校园跑腿核心业务需求的基础上，预留了智能派单、动态定价和数据分析等扩展能力，具备良好的可维护性与可扩展性。
